# ✨ Sílabas Mágicas

Projeto educacional desenvolvido como jogo interativo para auxiliar crianças no aprendizado de leitura através de desafios com sílabas.

## 📄 Descrição

Sílabas Mágicas é um jogo educativo web criado para crianças de aproximadamente 7 anos. O objetivo do jogo é estimular o aprendizado da leitura por meio de desafios interativos que envolvem a identificação e combinação de sílabas.

Durante o jogo, o usuário participa de diferentes fases onde precisa encontrar ou formar palavras utilizando sílabas corretas. Conforme o jogador avança, novas etapas são desbloqueadas, trazendo uma experiência progressiva e motivadora.

O sistema também conta com um chatbot com inteligência artificial que auxilia o jogador durante o aprendizado.

## 🎯 Objetivo do projeto

O projeto foi desenvolvido com foco em:

- estimular o aprendizado da leitura de forma divertida
- aplicar conceitos de desenvolvimento web
- criar uma experiência interativa para crianças
- integrar recursos de inteligência artificial em um ambiente educacional

## 🧩 Funcionalidades

- Jogo interativo de caça-sílabas
- Sistema de progressão por fases
- Narrativa educativa desbloqueada conforme o avanço do jogador
- Chatbot com inteligência artificial para auxiliar o usuário
- Sistema de recompensas com créditos obtidos durante o jogo
- Loja de skins para personalização do avatar do jogador
- Interface visual simples e intuitiva
- Aplicação acessível diretamente no navegador

## 🎁 Sistema de recompensas

Durante o progresso no jogo, o jogador recebe créditos ao completar desafios e fases.

Esses créditos podem ser utilizados para desbloquear **skins para o avatar**, permitindo que o jogador personalize sua aparência dentro do jogo.

Esse sistema foi desenvolvido para:

- aumentar o engajamento das crianças
- incentivar a progressão nas fases
- tornar a experiência mais divertida e personalizada

## 🎮 Como funciona

1. O jogador inicia o jogo diretamente pelo navegador.
2. Cada fase apresenta um desafio envolvendo sílabas.
3. O usuário deve identificar ou combinar sílabas corretamente.
4. Ao completar o desafio, novas fases são desbloqueadas.
5. O chatbot pode ajudar o jogador com orientações durante o jogo.
